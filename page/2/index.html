图片懒加载的简单实现

发表于 2017-05-25 | 分类于 javaScript

为什么要使用懒加载

在网页中，我们有时候需要展示多张图片。如果图片太多，页面的加载速度会慢很多，这样会非常影响用户体验。所以，在图片过多的页面，为了加速页面的加载进度，我们需要将页面中没出现在可视区域内的图片先不做加载，等图片滚动到可视区域内时，再把图片加载出来，这样对于页面的加载性能会有很大的提升。
